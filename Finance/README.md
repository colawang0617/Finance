# Finance Data Directory

此目录用于存放您的财务数据Excel文件。

## 所需文件

将您的Excel文件命名为 `财务跟踪表_完整版_KL.xlsx` 并放置在此目录中。

## 文件结构要求

Excel文件必须包含以下工作表：

### 1. 每日数据 (Daily Data)
- **第1行**: 表头
- **第2行**: 空行
- **第3行起**: 数据行

### 列结构 (A-R列):
- A: 日期 (MM-DD格式)
- B: 场地入账金额 (公式计算)
- C-H: 场地收入明细
- I: 云店销售 (公式计算)
- J-L: 商品销售明细
- M-P: 充值与卡类
- Q: 每日销售合计 (公式计算)
- R: 当日总计 (公式计算)

详细列映射请参考主 README.md 文件。

### 2. 月度汇总 (Monthly Summary)
自动从"每日数据"工作表计算生成的月度汇总。

### 3. 总计 (Total)
总体统计数据。

## 数据隐私

**注意**: `.gitignore` 已配置为忽略此目录中的所有 `.xlsx` 文件，确保您的财务数据不会被提交到版本控制系统。

## 备份文件

程序会自动创建备份文件，格式为：
```
财务跟踪表_完整版_KL_backup_YYYYMMDD_HHMMSS.xlsx
```

这些备份文件也会被 `.gitignore` 自动忽略。
